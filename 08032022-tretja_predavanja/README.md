# Omrezja predavanja

## Ponovitev

**Način zapisa signala** določa **fizični sloj**

Za ponovno **pošiljanje podatkov v primeru trčenja podatkov** je odgovoren **linijski sloj** 

Za usmirjanje je odgovoren **mrežni sloj**
